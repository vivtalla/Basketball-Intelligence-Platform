# Specs Index

All feature specs and agent handoff documents. Newest sprint first.

To write a new spec, copy `TEMPLATE.md` and name it:
- Handoff: `handoff-{from}-to-{to}-{slug}.md`
- Self-scoped: `sprint-{N}-{agent}-{slug}.md`
- Sprint closeout: `sprint-{NN}-closeout.md`

After creating a spec, add it here and add it to the Handoff Queue in `AGENTS.md`.
Sprint closeout notes are not handoffs; they are the shared end-of-sprint memory layer and should be updated once per sprint.
